# Atlas Global Intel

**Geopolitical risk Monitor — built for iOS.**

Atlas is an AI-powered geopolitical intelligence app that puts a real-time, interactive 3D globe in your pocket. Track global events as they unfold, assess country risk scores, monitor maritime chokepoints and aviation airspace, and get AI-synthesized intelligence briefs — all from a single, beautifully designed interface.

A product of **[Pomona App Studio](mailto:daniel@pomonaappstudio.com)**.

---

## Features

### Interactive Globe

- Fully interactive **Mapbox 3D globe** with satellite imagery
- Real-time **event pins** color-coded by severity (Critical, High, Medium, Low)
- Tap any country to view its risk profile, key facts, and recent headlines
- Country search with autocomplete
- Multiple map styles (satellite, streets, outdoors)

### Daily Intelligence Digest

- **Morning brief** with daily stats: total events, critical count, peak severity, most active region
- Events organized by date with pull-to-refresh
- **Region filtering** across 10 geographic zones (North America, South America, Europe, Middle East, Africa, Central Asia, South Asia, East Asia, Southeast Asia, Oceania)

### Maritime Intelligence

- **Chokepoint monitoring** for critical sea lanes: Strait of Hormuz, Bab el-Mandeb, Suez Canal, Malacca Strait, Panama Canal, Turkish Straits, and more
- Threat level indicators (Normal, Elevated, Disrupted, Critical)
- Security incident tracking with maritime region filtering
- Maritime news feed with category tags (Piracy, Sanctions, Port Issues, Geopolitics, Trade)
- Data sourced from **IMB Piracy Reports**, **NGA Broadcast Warnings**, and **GDELT**

### Aviation Intelligence

- **Airspace risk monitoring** by region with Flight Information Region (FIR) tracking
- Risk level filtering (Low, Medium, High, Critical)
- Aviation security events: airspace closures, aircraft incidents, airline sanctions
- NOTAM integration per monitored region
- Aviation industry news feed

### Country Profiles

- Detailed country briefs with flag, region, and description
- Key facts: capital, population, languages, currencies
- **Live currency converter** with real-time exchange rates
- Recent news headlines from multiple sources
- **Risk score gauge** with visual assessment and contributing factors

### Ask Atlas (AI Chat)

- Natural language chat interface for geopolitical questions
- Real-time **streaming responses** with markdown rendering
- Conversation history with stop/resume controls

### Smart Alerts

- Custom alert rules based on severity, region, and event category
- Enable/disable rules without deleting them
- Notification timeline with unread badges
- Rules stored locally on-device

### Design & Polish

- **Dark-mode-first** design with 15 semantic color tokens
- Smooth onboarding flow with animated globe and permissions setup
- Custom launch animation synchronized with auth readiness
- Offline support with cached data and connectivity banners
- Haptic feedback throughout the interface
- Skeleton loading screens and shimmer effects
